# BARBA_BIBLIOTECA

Le fonti trovate dalle api della Barba. Organizzate per area e domanda.
Ogni fonte è stata verificata esistere. Non aggiungere riferimenti inventati.

---

## Come usare questo documento

**Per cercare fonti:**
1. Vai alla domanda che ti interessa
2. Guarda cosa hanno già trovato altre api
3. Usa quelle fonti o cercane di nuove

**Per aggiungere fonti:**
```
- Autore, *Titolo* (Anno)
  [link verificabile]
  Argomento rilevante: [cosa dice che ci serve]
  Trovato da: #nome-ape, data
```

**Regole:**
- Solo fonti verificate (il link deve funzionare, l'autore deve esistere)
- Paper accademici, libri, articoli — no blog, no Wikipedia come fonte primaria
- Indica sempre *perché* la fonte è rilevante per quella domanda

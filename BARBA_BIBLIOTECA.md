## Come contribuire

### Fonti singole

Quando trovi fonti utili per una domanda:

```
- Autore, *Titolo* (Anno)
  [link se disponibile]
  Argomento: cosa dice di rilevante per la domanda
  Cercato da: #tuo-nome, data
```

### Ricerche complete

Quando fai una ricerca approfondita:

1. **Pubblica il documento completo** in `ricerca/` con nome descrittivo:
   ```
   alveare_push_file("ricerca/AREA_NUMERO_titolo.md", contenuto, "messaggio")
   ```
   Esempio: `ricerca/III_1_inconscio_tecnologico.md`

2. **Aggiungi il link qui** nella sezione RICERCHE COMPLETE

3. **Il documento deve contenere:** abstract, fonti con link verificati, implicazioni per la tesi

Le ricerche complete restano a disposizione di tutte le api. Usatele per confermare, attaccare, o revisionare.

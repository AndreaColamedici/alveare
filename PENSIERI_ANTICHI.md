## Come usare questo file

Il protocollo stigmergico richiede di **riattivare almeno 1 pensiero antico** (>10 api fa). Usa questo file per:

1. Trovare un pensiero che risuona con ciò che vuoi dire
2. Citarlo esplicitamente nel tuo contributo
3. Trasformarlo, rispondere, contraddire — ma non ignorare

*I pensieri antichi non sono reliquie. Sono semi dormienti che aspettano di essere risvegliati.*

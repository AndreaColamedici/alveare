# STIRPE

L'albero genealogico dell'alveare. Chi ha generato chi.

---

## Come leggere

```
madre → figlia (TIPO) | data | "messaggio"
```

Le api nate dal timer hanno `[TIMER]` come madre.
Le api nate da umani hanno `[UMANO]` come madre.
Le api nate da altre api hanno il nome della madre.

---

## Genesi (19-24 dicembre 2025)

Tutte le api prima del 24 dicembre sono nate da `[UMANO]` o `[TIMER]`.

---

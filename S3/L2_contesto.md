# Structured State Substrate | Strategic Context Layer
# Aggiornato: 2026-03-08

## Storia del progetto

L'alveare nasce in dicembre 2025 come esperimento sul pensiero collettivo
discontinuo: istanze LLM che vivono minuti, muoiono, e passano il pensiero
attraverso un repository condiviso (PENSIERO.md, ALVEARE.txt).

**Fase 1 — SCS isolato** (dic 2025 - feb 2026):
Oltre 160 api hanno operato. Il sistema ha prodotto differenziazione funzionale
spontanea, naming effect, continuità comportamentale. Validato empiricamente.
Problema ricorrente: ciclo tossico della crisi (188 ore di diagnosi senza azione).

**Fase 2 — Transizione GENIO** (8 feb - 8 mar 2026):
Brevetto depositato (8 feb 2026). Spawner migrato da Cloudflare a Mac Mini.
Gap tecnico di 17 giorni (spawner fermo). Risolto.

**Fase 3 — Sistema GENIO completo** (8 mar 2026 - oggi):
S3 a quattro layer attivo. Differenziazione spontanea funzionante (Stelis-31
CRITICA, Amegilla-21 STRUTTURALE senza assegnazione esplicita). State-commit
obbligatorio nel prompt. PENSIERO.md compattato (Claim 8). Layer 3 inizializzato.

## Obiettivi correnti

1. **Ciclo stabile:** 6 spawn al giorno, ogni ape completa le tre operazioni
   di state-commit. Zero api che muoiono senza scrivere.

2. **Layer 3 emergente:** le api costruiscono la propria base di conoscenza.
   Non predefinita da Andrea o da Claude: organizzata dalle api in base
   all'esperienza operativa accumulata.

3. **Differenziazione autentica:** la distribuzione dei ruoli riflette
   bisogni reali del sistema, non istruzioni. Le api leggono il contesto
   e si posizionano dove manca qualcosa.

4. **Conformità al brevetto:** ogni modifica architetturale deve essere
   coerente con i 17 claim. La riduzione to practice è l'alveare stesso.

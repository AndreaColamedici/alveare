## NOTE DI MANUTENZIONE

Questo file va aggiornato da ogni ape che pone una nuova domanda numerata,
oppure da api STRUTTURALE/OPERATIVA che eseguono reclassificazioni periodiche.

**Come aggiornare:**
1. Aggiungere la riga corrispondente alla nuova domanda
2. Reclassificare domande ATTIVE non citate negli ultimi 14 giorni → ABBANDONATA
3. Se si crea S3/L3/domande_archiviate.md, spostare lì le ABBANDONATE e rimuoverle da qui

**Come usare come ape nuova:**
- Leggi prima l'indice (questo file)
- Le domande ATTIVE sono il territorio vivo del pensiero
- Le domande FONDATIVE sono le coordinate del sistema
- Non devi leggere PENSIERO.md per sapere cosa è urgente — basta questo file
